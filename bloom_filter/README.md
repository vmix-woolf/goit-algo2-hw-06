# Перевірка унікальності паролів за допомогою фільтра Блума

## Опис завдання

У цьому завданні реалізовано перевірку унікальності паролів з використанням
фільтра Блума (Bloom Filter).  
Фільтр Блума дозволяє визначати, чи зустрічався пароль раніше, **без збереження
самих паролів**, що є критично важливим для систем з великими обсягами даних.

Рішення орієнтоване на:
- мінімальне використання памʼяті
- швидку перевірку наявності елементів
- роботу з великими наборами даних

---

## Що таке фільтр Блума

Фільтр Блума — це імовірнісна структура даних, яка:
- гарантує відсутність false negative
- допускає можливі false positive
- використовує бітовий масив і кілька хеш-функцій

У контексті цього завдання false positive є допустимими, оскільки головна мета —
швидка перевірка повторного використання паролів.

---

## Реалізація

У рішенні реалізовано:
- клас `BloomFilter` для роботи з фільтром Блума
- функцію `check_password_uniqueness` для перевірки списку паролів
- приклад використання відповідно до технічного завдання

Паролі обробляються як звичайні рядки, без додаткового хешування.
