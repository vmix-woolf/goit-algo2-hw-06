# Порівняння точного підрахунку та HyperLogLog

## Опис

У цьому завданні виконано порівняння двох підходів до підрахунку
унікальних IP-адрес у лог-файлі:

- точний підрахунок з використанням структури `set`
- наближений підрахунок з використанням алгоритму HyperLogLog

Метою є демонстрація різниці у швидкодії та точності для великих наборів даних.

---

## Реалізація

Скрипт:
- построчно обробляє лог-файл
- ігнорує некоректні рядки
- виконує точний та наближений підрахунок
- вимірює час виконання кожного методу
- виводить результати у вигляді таблиці

---

## Запуск

Запускати з кореня проєкту:

```bash
python hyperloglog/main.py
```
